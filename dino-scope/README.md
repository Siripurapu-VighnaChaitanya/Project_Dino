## Getting Started

### Prerequisites

- Node.js 16+ and npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Siripurapu-VighnaChaitanya/Project_Dino.git

# Navigate to project directory
cd Project_Dino

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
